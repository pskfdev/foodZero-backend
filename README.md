# website foodZero (CRUD)

เป็นโปรเจคที่จัดทำขึ้นเพื่อเป็นผลงานในการฝึกเขียนเว็บไซต์ในส่วนหลังบ้าน โดยใช้ `Express`

มีฟังก์ชั่นดังนี้

* Login / Logout
* CRUD User
* CRUD Product
* CRUD Category
* CRUD Reservation
* CRUD Blog

## language & library

* Express
* bcryptjs
* body-parser
* cors
* jsonwebtoken
* mongoDB
* mongoose
* morgan
* multer
